## Usage

[Download the latest Alpha release!](https://github.com/arthurdevv/_alpha/releases/latest)

Alpha will automatically update when a new release is available.

## Features

- Cross platform
- User settings
- Font ligatures
- Unicode support
- Supports CMD, PowerShell, Git-Bash, WSL, Cygwin
